# SQL_Aggregate_Functions_And_Operators
This repository contains my second SQL fundamentals exercise, focusing on aggregate functions and data summarisation. The project builds on basic SQL skills and introduces grouping, filtering aggregated data, and applying logical operators.

## 📌 Project Overview

The goal of this exercise was to develop a deeper understanding of how to summarise and analyse data using SQL.

The exercise includes 15 practical questions across multiple datasets:
- students  
- courses  
- enrollments  
- salaries  
- projects  

Each question required:
- Writing accurate SQL queries  
- Applying aggregate functions  
- Using grouping and filtering logic  
- Predicting the expected output  

## 🧠 Key Skills Demonstrated

- Using aggregate functions: COUNT, SUM, AVG, MIN, MAX  
- Grouping data with GROUP BY  
- Filtering grouped results using HAVING  
- Applying SQL operators: DISTINCT, BETWEEN, IN, NOT, AND, OR  
- Aliasing results using AS  
- Understanding the difference between WHERE and HAVING
